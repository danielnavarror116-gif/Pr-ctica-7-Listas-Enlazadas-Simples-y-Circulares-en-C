# Pr-ctica-7-Listas-Enlazadas-Simples-y-Circulares-en-C

Preguntas-Lista Simple
1. ¿Por qué las listas enlazadas no requieren tamaño fijo como los arreglos?
   Porque los nodos se crean dinámicamente con malloc() solo cuando se necesitan.
   
2. ¿Qué diferencia hay entre mover la cabeza y modificar los punteros internos?
   Al mover la cambeza solo cambia el puntero y al modificar los punteros interson se podria decir que cambia la lista por dentro.
   
Preguntas-Lista Circular
1. ¿Qué ventaja tiene la lista circular frente a la simple?
   Como no tal no tiene final por q el ultimo nodo regresa al primero
   
2. ¿Qué error puede causar un bucle infinito en una lista circular?'
   Que el programa se vaya al infinito
  
3. ¿Qué estructuras o programas del mundo real se basan en listas circulares?
   Cuando ponemos por ejemplo una playlist en spotify y hacemos q entre en bucle, incluso se podria en juegos de mesa como el monopoly q es por turnos.
